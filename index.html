Hello everyone,

I'm sorry to say that AtticRadar is shutting down.
I don't have the same motivation & ideas for this website that I used to.
Also, a lot of the code would need a complete rewrite to make future development go smoothly.

Some might wonder why I have to shut it down instead of just letting the website continue running.
I was starting to accrue significant charges from my map tiling provider because enough people were using AtticRadar monthly.
This, coupled with server management and costs, made it a massive headache to manage all these issues on a "legacy" project like this.

The source code will remain on GitHub for those interested.

Thanks for all the support over the past few years! You all have been awesome to me.
